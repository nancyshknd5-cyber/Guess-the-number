# Guess-the-number
A simple Python mini project where the player guesses a randomly generated number between 1 and 100. The game provides hints after each guess and ends when the correct number is guessed or the player quits.
# Number Guessing Game (Python)

This is a simple Python command-line based number guessing game.
The program generates a random number between 1 and 100, and the user
tries to guess the correct number.

## Features
- Random number generation using Python's `random` module
- User-friendly hints for higher or lower guesses
- Option to quit the game anytime
- Beginner-friendly Python project

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run the file using:
   python guess_number.py

## Learning Outcomes
- Understanding loops and conditional statements
- Using Python modules
- Handling user input
- Basic game logic implementation
